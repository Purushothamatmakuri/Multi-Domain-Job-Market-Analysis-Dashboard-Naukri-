## 📌 Project Overview

This project analyzes **real-time job market trends in India** by scraping job postings from Naukri and transforming raw data into **actionable insights** through a Power BI dashboard.

👉 The goal was to simulate a **real-world data analytics workflow**:

* Data Collection (Web Scraping)
* Data Cleaning & Transformation
* Exploratory Data Analysis (EDA)
* Business Intelligence Dashboarding

---

## 🎯 Business Problem

Recruiters and job seekers often lack **structured insights** into:

* Which roles are in demand
* What skills are required
* Where hiring is concentrated

This project solves that by converting **unstructured job listings into structured intelligence**.
